# Session Handoffs Archive

This folder stores historical `SESSION_HANDOFF_*` files moved from repository root.

## Policy

- Keep only active handoff files in repository root.
- Move completed or outdated handoffs to this archive folder.
- Do not delete archived handoffs unless there is a separate retention decision.

## Current Root Convention

Keep only one active handoff in repository root (the current session context). Move older files to this archive.

Example:

- `SESSION_HANDOFF_2026-03-31_UI_RESTORE_STATUS.md`
